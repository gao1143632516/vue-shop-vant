## vue + vant + vue-router + vuex

>此项目只有前端项目，后端接口没有出来，仅供大家参考

-支付宝授权登录
-支付宝网页支付
-购物车
-商品展示

## 快速上手

```
# 安装 Vue Cli 3
npm install -g @vue/cli

npm install

npm run serve

npm run build
```

调整src/config/env.js的配置信息
```
baseUrl: 域名地址
dataSources：数据源(local=本地)
```
